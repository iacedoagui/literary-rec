# literary-rec

A dashboard for academics: enter a topic or research idea and get recommended
pre-prints from **arXiv** and the **Semantic Scholar** API, automatically
**clustered into sub-topics**, each paper with a **toggleable one-paragraph
summary** generated by Claude.

## Architecture

```
React + TS (Vite)            FastAPI (Python)
─────────────────            ──────────────────────────────────────
search box       ──POST /api/recommend──▶  fetch arXiv + Semantic Scholar
cluster sections                            ├─ de-duplicate
paper cards      ◀─────────────────────     ├─ embed abstracts (sentence-transformers)
                                            ├─ cluster (KMeans, auto-k by silhouette)
                                            └─ label clusters (Claude)
expand a card    ──POST /api/summarize──▶  summarize one paper (Claude)
```

- **Clustering** runs on local sentence-transformer embeddings — no embedding
  API key needed. The cluster count is chosen automatically.
- **Summaries are lazy**: a paper is only sent to Claude when its card is first
  expanded, so a search stays fast and you only pay for summaries you read.
- **Model**: defaults to `claude-opus-4-8`. Set `SUMMARY_MODEL` / `LABEL_MODEL`
  to `claude-haiku-4-5` for cheaper, faster generation.

## Prerequisites

- Python 3.14
- Node 18+
- An Anthropic API key

## Backend

```bash
cd backend
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env          # then set ANTHROPIC_API_KEY
uvicorn app.main:app --reload --port 8000
```

The first request downloads the sentence-transformer model (~80 MB), so it is
slower than subsequent ones.

Health check: `curl localhost:8000/api/health`

## Frontend

```bash
cd frontend
npm install
npm run dev
```

Open http://localhost:5173. The Vite dev server proxies `/api` to the backend
on port 8000.

## Configuration

All backend settings are environment variables — see `backend/.env.example`.

| Variable                  | Default                                          | Purpose                              |
| ------------------------- | ------------------------------------------------ | ------------------------------------ |
| `ANTHROPIC_API_KEY`       | —                                                | Required for summaries & labels      |
| `SUMMARY_MODEL`           | `claude-opus-4-8`                                | Per-paper summary model              |
| `LABEL_MODEL`             | `claude-opus-4-8`                                | Cluster-labeling model               |
| `EMBEDDING_MODEL`         | `sentence-transformers/all-MiniLM-L6-v2`         | Abstract embeddings                  |
| `MAX_CLUSTERS`            | `8`                                              | Upper bound on sub-topics            |
| `MAX_RESULTS_PER_SOURCE`  | `25`                                             | Papers fetched per source            |
| `SEMANTIC_SCHOLAR_API_KEY`| —                                                | Optional, raises S2 rate limits      |
| `ALLOWED_ORIGINS`         | `http://localhost:5173`                          | CORS origins                         |
