# literary-rec
A Dashboard for researchers to input their research interest and it will recommend existing literature in ArXiv creating summaries and clustering them by topic with the help of Claude Code

# Why?
I built this project as a tooling device for researchers to facilitate the process of doing a literary review. This process can be a bit overwhelming for an unexperienced researcher or tiring for an experienced one.

With this type of tool to use, they can see what currently exist around the topic of their research and also what other things, researchers around the world are currently doing.
